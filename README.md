# Backups Checker
### Automatic checker your backups


* Requirements
  * Python 3
  * Config Parser 3.7.1
  * Install: `pip install -r requirements.txt`
  
* Install
  * Download
  * Setup `Config/config`
  * Manual start `py main.py`
  * Automatic start: https://opensource.com/article/17/11/how-use-cron-linux
